Solusi Input Manual NFC Scanner

Extension browser Microsoft Edge/Chrome untuk menambahkan fitur input UID manual pada sistem loket MeRS saat perangkat NFC tidak tersedia.

Mulai Instalasi Cara Penggunaan

Fitur Utama

Didesain khusus untuk kebutuhan operasional di lapangan

⌨️

Manual Input Panel

Tombol melayang (FAB) di pojok kanan bawah membuka panel input manual. Solusi saat reader NFC bermasalah.

🔗

Quick Shortcuts

Akses cepat ke halaman Cek Menu dan Ambil Pesanan langsung dari dalam panel extension.

🕐

Smart History

Menyimpan 5 UID terakhir yang digunakan secara lokal. Memudahkan input ulang tanpa mengetik.

Panduan Instalasi

Microsoft Edge / Google Chrome

1

Buka Menu Extensions

Ketik edge://extensions di address bar browser Anda.

2

Aktifkan Developer Mode

Cari toggle Developer mode (biasanya di pojok kiri bawah atau kanan atas) dan aktifkan.

3

Load Unpacked

Klik tombol Load unpacked, lalu pilih folder MRS-NFC-Manual-Input yang sudah Anda download.

4

Selesai! 🎉

Extension akan aktif otomatis di halaman MeRS. Refresh halaman web untuk mulai menggunakan.

Cara Penggunaan

🍽️
cek_menu.html

Halaman Cek Menu

Digunakan untuk mengecek jatah makan karyawan.

  • Buka halaman via shortcut extension.
  • Klik tombol ⌨ (Keyboard) di pojok kanan bawah.
  • Masukkan UID kartu dan tekan Enter/Cek.
📲
nfc_scanner.html

Halaman Ambil Pesanan

Digunakan di loket pengambilan makanan.

Penting:

Anda HARUS MEMILIH LOKET terlebih dahulu sebelum melakukan input manual. Jika tidak, sistem akan menolak input.

  • Pilih Loket (1-6) di layar utama.
  • Setelah masuk mode scan, klik tombol manual input.
